# Task 3 - Matrix Rotate Transponse

```
Към класа Matrix да се добави функция rotate, която завърта масива на 90 градуса по часовниковата стрелка.

По желание: да се напише друга функция, която завърта в обратната посока.
Да се добави функция transponse, която транспонира матрицата. 
Това означава редовете да станат колони и обратното.

Пример rotate:
1 2 3  => 4 1
4 5 6     5 2
          6 3

Пример transponse:
1 2 =>  1 3
3 4     2 4
```
