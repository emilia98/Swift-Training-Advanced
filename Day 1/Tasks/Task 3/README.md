# Task 3 - Circle Buffer

```
Кръгов буфер

Да се разработи клас, който симулира работата на кръгов буфер. той работи така:
- данните се съхраняват в масив;
- класа има променлва (индекс), който показва до коя позиция има въведени данни;
- когато се добавят нови данни, те се добавят на позицията на индекса, 
  като индекса се премества така, че да сочи към първия свободен елемент след добавените данни;
- ако индекса прехвърли дължината на масива, той се премества на 0-ва позиция и данните се добавят от там нататък.

Пример:
нови данни: [10, 11, 12]

преди добавяне:
- индекс: 3
- буфер: [5, 0, 0, 0, 0]
- дължина на буфера: 5

след добавяне
- индекс: 1
- буфер: [12, 0, 0, 10, 11]

класа да има 2 функции
- write с параметър масив
- read - връща масив с данните постъпили след последното четене
```
