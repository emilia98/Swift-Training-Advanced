# Task 2 - Noice Reducer

```
Изглаждане на данни

В някои случаи, данните, идващи от даден сензор, са примесени с шум.
Примерно, ако температурата в стаята е 22 градуса, отчетите през 1 секунда са:
21, 22, 20, 24, 18, 21, 19 и т.н.

Да се напише функция, която елеминира шума чрез изчисляване на rolling/moving average. 
Това означава, че при подаване на нова стойност, 
функцията връща средно аритметичното на последните n стойности, 
като n се конфигурира (задава се като константа в кода).
```
