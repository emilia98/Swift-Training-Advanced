# Group Task 1 - Shortest Sequence

```
Group Task - Emilia, Dimitar & Georgi

Да се напише функция, която намира най-късата поредица от цифри в масив, 
която съдържа 3 зададени числа.

Пример:
[1, 3, 8, 4, 2, 8,  2, 11, 1, 5, 13],  [1, 4, 8] -> [1, 3, 8, 4]
Другата е [4, 2, 8,  2, 11, 1], но е по-дълга.
```
