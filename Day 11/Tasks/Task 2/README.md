# Task 2 - Polish Expression

```
Изчисление на израз, записан в обратен полски запис.
Да се напише функция, която изчислява израз, записан в обратен полски запис, който получава като стринг.
Характерно за обратния полски запис е, че операцията е записана след операндите.

Пример:
2 3 + => 5
3 1 - => 2

Този начин на записване елеминира необходимостта от скоби, когато трябва да се извършат няколко изчисления.
5 3 2 * - => 5 6 - => -1

Функцията може да се имплементира най-лесно, ако се ползва стек.
```
